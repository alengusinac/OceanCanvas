# OceanCanvas

## How to start

### Backend

cd backend/  
npm install  
nodemon start  

### Frontend

cd frontend/  
npm install  
npm run dev  

#### Run Cypress Tests

cd frontend/  
npx cypress run  

## Live Version

https://oceancanvas-frontend.onrender.com/

The API is on a free web service and may take up to a minute to start, please be patient and refresh the products page!
